# MIG4005 — EventLog

| | |
| --- | --- |
| **Category** | Runtime failures |
| **Severity** | Medium |
| **Effort** | Small |
| **Confidence tier** | 2 — Probable |

## What it detects

Use of the `EventLog` type (`System.Diagnostics.EventLog`).

## Why it matters

The Windows Event Log is Windows-only on modern .NET; `EventLog` throws
`PlatformNotSupportedException` off Windows.

## How to fix it

Use a cross-platform logging framework (e.g. `Microsoft.Extensions.Logging` with a suitable
sink) instead of writing directly to the Event Log.

## Known false positives

Tier 2. Matches the identifier `EventLog`, so an unrelated type named `EventLog` would also
match. Confirm it is `System.Diagnostics.EventLog`.
