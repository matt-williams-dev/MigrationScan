# MIG3003 — ASMX web service

| | |
| --- | --- |
| **Category** | Blocking frameworks |
| **Severity** | High |
| **Effort** | Large |
| **Confidence tier** | 1 — Certain |

## What it detects

The presence of an `.asmx` file — the signature of a legacy ASP.NET (ASMX) web service.

## Why it matters

ASMX has no counterpart on modern .NET. The service must be re-implemented, and clients that
consumed the SOAP endpoint may need updating too.

## How to fix it

Re-implement the service as an **ASP.NET Core Web API** (REST/JSON) or **gRPC**. If SOAP
compatibility is required, consider CoreWCF.

## Known false positives

None. The presence of an `.asmx` file is unambiguous.
