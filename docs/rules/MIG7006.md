# MIG7006 — LINQ to SQL (System.Data.Linq)

| | |
| --- | --- |
| **Category** | Data access |
| **Severity** | High |
| **Effort** | Large |
| **Confidence tier** | 2 — Probable |

## What it detects

Use of the `System.Data.Linq` namespace (LINQ to SQL).

## Why it matters

LINQ to SQL is not available on modern .NET and has no direct port. The data layer must be
re-implemented.

## How to fix it

Migrate to **Entity Framework Core** or another supported ORM. Expect to rewrite the data
context and mapping; queries often port with edits.

## Known false positives

Low. The namespace is specific. Detection matches imports and fully-qualified references.
