# MIG3009 — MSMQ (System.Messaging)

| | |
| --- | --- |
| **Category** | Blocking frameworks |
| **Severity** | High |
| **Effort** | Large |
| **Confidence tier** | 2 — Probable |

## What it detects

Use of the `System.Messaging` namespace (MSMQ).

## Why it matters

`System.Messaging` is not available on modern .NET. Message-queue code built on MSMQ must move
to a supported queue.

## How to fix it

Migrate to a supported message queue — **Azure Service Bus**, **RabbitMQ**, or similar — or a
third-party MSMQ-compatible library if you must keep talking to existing MSMQ infrastructure.

## Known false positives

Low. The namespace is specific. Detection matches imports and fully-qualified references.
