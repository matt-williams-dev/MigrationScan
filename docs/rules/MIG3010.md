# MIG3010 — ASP.NET MVC 5 (System.Web.Mvc)

| | |
| --- | --- |
| **Category** | Blocking frameworks |
| **Severity** | High |
| **Effort** | Large |
| **Confidence tier** | 2 — Probable |

## What it detects

A `using System.Web.Mvc;` directive — the ASP.NET MVC 5 namespace, which builds on
`System.Web` and runs only on .NET Framework.

## Why it matters

`System.Web.Mvc` is not available on modern .NET. The controller/view model carries over to
ASP.NET Core MVC, but the host, routing, filters, and DI wiring differ.

## How to fix it

Migrate to ASP.NET Core MVC (`Microsoft.AspNetCore.Mvc`). Controllers and Razor views port
with edits; `Global.asax`, `HttpContext`, and `System.Web`-based filters need rework.

## Known false positives

Low. `System.Web.Mvc` is specific to MVC 5. Detection matches both `using` directives and
fully-qualified references.
