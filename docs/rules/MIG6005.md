# MIG6005 — Obsolete cryptography types

| | |
| --- | --- |
| **Category** | Serialization and security |
| **Severity** | Medium |
| **Effort** | Small |
| **Confidence tier** | 2 — Probable |

## What it detects

Use of concrete cryptography implementation types that are obsolete on modern .NET —
`*CryptoServiceProvider` (e.g. `RNGCryptoServiceProvider`, `MD5CryptoServiceProvider`) and
`*Managed` (e.g. `SHA1Managed`, `RijndaelManaged`).

## Why it matters

These types are deprecated on modern .NET (they raise obsoletion warnings), and several wrap
weak algorithms (MD5, SHA1, DES) that should not be used at all.

## How to fix it

Create algorithms via the base-class factories — `SHA256.Create()`, `Aes.Create()`,
`RandomNumberGenerator` — and drop weak algorithms entirely.

## Known false positives

Tier 2. Matches the type names by identifier. A same-named type of your own would also match.
